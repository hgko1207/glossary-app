## 용어 사전

Spring Boot, React 및 Ant Design을 사용하여 용어 사전 정리하는 앱 빌드

### 기능

- 한글 단어, 동사 및 용어를 조회
- 영어 문자와 용어에 대한 설명을 출력
- 용어 사전을 통해 개발 시 용어 통일화를 할 수 있도록
